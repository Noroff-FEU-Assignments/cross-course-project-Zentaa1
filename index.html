Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

I would recommend to use a SSL certificate so that your sensitive information will become encrypted (impossible to read).
You should also back up your site regularly so if someone breaches the site and messes with the code you have a backup laying around.
Also if you are using or planning on using third party apps/softwares, always make sure they are up to date and remove the apps you don't use, Also research everywhere if the app you are going to use is even safe to use.
You should always check on sites like "https://pentest-tools.com/website-vulnerability-scanning/website-scanner" if there is any major security issues on your site.
Continuous Deployment/Continuous Integration tools are also very handy when it comes to web security since it automatically runs every change in code trough different security checks and will decline any changes that does not get approved by the checks, the codes that gets declined will be flagged for a rework so you have to rewrite the code.
I would also highly recommend using a Load Balancer since that can avoid some DDoS attacks, a Load balancer would "sit in front" and route client requests across all servers so that one server wont become overloaded.
